 ShopEZ - Stock Trading Platform

## Project Overview
ShopEZ is a full-stack web application built using Node.js, Express, and MongoDB.  
It allows users to view products and manage data using backend APIs and frontend UI.

## Features
- View products from database
- REST API using Express.js
- MongoDB integration
- Frontend + Backend full stack project

##  Tech Stack
- HTML, CSS, JavaScript
- Node.js
- Express.js
- MongoDB

## Project Structure
- client/ → Frontend
- models/ → Database schemas
- routes/ → API routes
- server.js → Backend server
- 
## How it works
Frontend sends request → Backend API → MongoDB → Response returned → UI displays data
